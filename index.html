Codecademy HTML/CSS Cheatsheet

HTML

HTML or HyperText Markup Language is a markup language used to display documents in a web browser.

Basic HTML tags
Tag Description Common use
<!DOCTYPE> Tag to specify the document type. This tag has to be placed at the very begining of your html document.
<html> HTML code container. This tag is used to contain your html code within your document.
<head> Contains the metadata. Typically contains the document title, character set, styles, scripts, etc.
<body> Defines the document body. This tag is used to contain all the contents of the document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<footer> Defines the document footer section. This tag is used to contain information, such as authorship, copyright, contacts, etc.

CSS

CSS or Cascading Style Sheets is a style sheet language used for formating the presentation of a document written in a markup language such as HTML or XML

CSS Ruleset Terms
Term exemple Description
Selector P { color: blue; } The beginning of the ruleset, targets the element to be styled.
Declaration Block P { color: blue; } The code in-between (and including) the curly braces ({ }) that contains the CSS declaration(s).
Declaration P { color: blue; } The group name for a property and value pair that applies a style to the selected element.
Property P { color: blue; } The first part of the declaration that signifies what visual characteristic of the element is to be modified.
Value P { color: blue; } The second part of the declaration that signifies the value of the property.
